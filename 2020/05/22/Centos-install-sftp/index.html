CentOS install sftp

思路

  1. 创建用户，创建用户组，添加用户到用户组,禁止用户登录。
  2. 创建用户登录的目录
  3. 设置用户的密码
  4. 修改ssh的配置文件，限制用户访问根目录,强制用户使用internal-sftp，指定用户主目录
  5. 设置目录权限，所属主和组，所有者须为root，用户组为sftp
  6. 重启ssh服务

测试

  1. 在Windows上cmd命令：sftp sftpuser@domain
  2. 即可访问目录/文件
