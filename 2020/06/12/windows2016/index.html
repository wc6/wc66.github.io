环境准备

  1. Windows server 2016

    tips: 我是在腾讯云上租赁的一台服务器，现在重置系统为Windows Server 2016。

  2. SQL Server 2016

    tips: 产品密钥参见：点击打开链接

  3. SSMS SQL Server Management Studio sqlserver 集成工具

    tips: 微软官方下载链接

  4. SSMS 是SQL server集成管理工具

  5. 友情提示：sql server 工具和链接器

MSSQL 2016安装步骤

  1. 先安装SQL server ，双击虚拟光驱加载iso文件，再点击setup文件即安装。

  1. 全新安装

  2. 安装过程

  3. 接受协议，下一步。

  4. 然后中途就会到这个界面。(我选择安装时更新。)

  5. 上面一个步骤完成之后，经过短暂的等待，就进入功能选择界面，这里选择下面这些功能：

    数据库引擎服务 > SQL Sserver 复制

    共享功能 > 客户端工具连接

    共享功能 > 客户端工具向后兼容性

    共享功能 > 客户端工具SDK

    共享功能 > SQL客户端连接SDK

    如果C盘空间不足，可以设置为D盘

  6. 一直下一步，直到身份认证模式，

    设置密码，强度要达标， 和Windows用户密码设置规则一样。

    设置SQL server管理员用户。

  7. 安装概要,确认安装信息，点击安装。

  8. 安装完成
